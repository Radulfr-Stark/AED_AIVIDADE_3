# AED_AIVIDADE_3
Repositório para implemento da Atividade 3 da disciplina AED_Engenharia_Software.

Jogo de palavras onde o jogador tenha que adivinhar uma palavra que será mostrada na tela com as letras embaralhadas.
1. Utilizar funções;
2. Sem uso do método *shuffle()*;
3. As lista de palavras possíveis deverá ser preenchida pelo usuário;
4. O programa deverá escolher uma palavra da lista aleatoriamente (função);
5. A palavra será exibida em tela de forma embaralhadas em suas letras (utilizar passagem de parâmetro). 
6. O jogador terá sete tentativas para advinhar a palavra (recebendo a resposta por parâmetro e retornando True ou False)
